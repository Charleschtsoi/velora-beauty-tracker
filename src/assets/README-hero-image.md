# Hero banner image

Place your hero image (the skincare/floral image shown under the VELORA title on the home screen) as:

**`hero-banner.png`** in this folder (`src/assets/`).

Supported formats: PNG or JPG. If you use JPG, rename the file to `hero-banner.jpg` and in `HomeScreen.tsx` change the require to `require('../assets/hero-banner.jpg')`.

Recommended size: about 560×240 px (or similar aspect ratio) so it looks sharp on high-DPI screens.
